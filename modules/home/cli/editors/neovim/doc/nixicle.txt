INTRODUCTION                                                   *nixicle-intro*

This is a personal Neovim configuration built with nixCats and lze for lazy
loading. The configuration is managed via Nix and supports both NixOS and
non-Nix environments.

Repository: ~/nixicle/modules/home/cli/editors/neovim

LEADERS                                                      *nixicle-leaders*

<leader>     = <Space>
<localleader> = ,

NAVIGATION                                                *nixicle-navigation*

Basic Movement~
  j/k              Move up/down (wraps on soft-wrapped lines)
  <C-d>/<C-u>      Half-page down/up (keeps cursor centered)
  n/N              Next/previous search (keeps cursor centered)

Buffer Navigation~
  <leader><leader>[ Previous buffer
  <leader><leader>] Next buffer
  <leader><leader>l Last (alternate) buffer
  <leader><leader>d Delete current buffer

Split Navigation~
  <C-h>            Move to left split
  <C-j>            Move to down split
  <C-k>            Move to up split
  <C-l>            Move to right split
  <C-\>            Move to previous split

Split Swapping~
  <leader><leader>h Swap buffer left
  <leader><leader>j Swap buffer down
  <leader><leader>k Swap buffer up
  <leader><leader>l Swap buffer right

Quickfix/Location Lists~
  <C-n>            Next quickfix item
  <C-p>            Previous quickfix item
  <leader>j        Previous location list item
  <leader>k        Next location list item
  <leader>q        Toggle quickfix (quicker.nvim)
  <leader>l        Toggle location list (quicker.nvim)
  <leader>Q        Toggle quickfix without focus
  >                Expand quickfix context (in quickfix window)
  <                Collapse quickfix context (in quickfix window)

FILE MANAGEMENT                                              *nixicle-files*

File Explorer~
  <leader>e        Toggle file explorer (mini.files)
  -                Open oil.nvim (parent directory)
  <leader>r-       Open nvim root directory (oil.nvim)

Telescope~
  <leader>ff       Find files
  <leader>fg       Live grep
  <leader>fb       Find buffers
  <leader>fh       Find help
  <leader>fm       Find keymaps
  <leader>fs       Find telescopes
  <leader>fw       Find current word
  <leader>fd       Find diagnostics
  <leader>fr       Find resume
  <leader>f.       Find recent files
  <leader>fc       Find command history
  <leader>fn       Find in notes directory
  <leader>fN       Find in Nix config
  <leader>p        Yank history (yanky)

Scratch Buffers~
  <leader>.        Toggle scratch buffer (snacks.nvim)
  <leader>S        Select scratch buffer
  q                Close scratch buffer (in scratch window)
  <Esc>            Close scratch buffer (in scratch window)

Session Management~
  Automatically saves and restores sessions per directory

LSP & CODE                                                      *nixicle-lsp*

LSP Commands~
  gd               Go to definition
  gr               Go to references
  gI               Go to implementation
  <leader>D        Type definition
  <leader>ds       Document symbols
  <leader>ws       Workspace symbols
  K                Fancy LSP hover (OXY2DEV)
  gK               Signature help

Diagnostics~
  [d               Previous diagnostic
  ]d               Next diagnostic
  gD               Fancy diagnostics hover (OXY2DEV)
  <leader>cd       Open diagnostic float
  <leader>q        Diagnostics to location list
  q                Close hover/diagnostic window (when focused)

Code Actions~
  <leader>ca       Code actions (tiny-code-action)
  <leader>rn       Rename symbol (inc-rename)
  <leader>rE       Inline edit

Refactoring~
  <leader>re       Extract (visual mode)
  <leader>rf       Extract to file (visual mode)
  <leader>rv       Extract variable (visual mode)
  <leader>ri       Inline variable
  <leader>rI       Inline function
  <leader>rb       Extract block
  <leader>rbf      Extract block to file

Formatting & Linting~
  Formatting happens via conform.nvim on save
  Linting happens via nvim-lint automatically

SEARCH & REPLACE                                             *nixicle-search*

Search~
  <Esc>            Clear search highlight
  /                Search forward
  ?                Search backward

Replace~
  <leader>sr       Open grug-far (search & replace)
  <leader>sw       Replace current word
  <leader>sp       Replace in current buffer (visual mode)

NOTES & DOCUMENTATION                                         *nixicle-notes*

Zk (Zettelkasten Notes)~
  <leader>zn       Create new note
  <leader>zo       Open notes picker
  <leader>zt       Search by tags
  <leader>zf       Search notes content
  <leader>znt      New note from title (visual)
  <leader>znc      New note from content (visual)
  <leader>zi       Insert link to note

Img-clip~
  <leader>ip       Paste image from clipboard

Markview~
  Markdown preview enabled automatically for .md files
  YAML frontmatter with custom icons (id, tags, etc.)

Features~
  - Live markdown preview
  - Hybrid mode (edit and preview simultaneously)
  - Callout support
  - Checkbox rendering
  - Math support (LaTeX)

==============================================================================
WINDOW MANAGEMENT                                            *nixicle-windows*

Split Creation~
  <leader>|        Split window right
  <leader>-        Split window below

General~
  <C-s>            Save file
  <leader>mj       Move line down
  <leader>mk       Move line up
  J                Join line (visual mode moves selection)
  K                Move selection up (visual mode)

Indentation~
  >                Indent right (stays in visual mode)
  <                Indent left (stays in visual mode)

Editing~
  <C-p>            Paste from clipboard (insert mode)
  J                Join line (keeps cursor position)

Miscellaneous~
  <leader>u        Toggle undotree
  <leader>tt       Open current file in new terminal

==============================================================================
FILE BOOKMARKS                                              *nixicle-bookmarks*

Warp.nvim (similar to harpoon)~
  <leader>ha       Add file to warp
  <leader>he       Show warp list
  <leader>hd       Remove file from warp
  <leader>1        Go to warp file 1
  <leader>2        Go to warp file 2
  <leader>3        Go to warp file 3
  <leader>4        Go to warp file 4
  <leader>hn       Next warp file
  <leader>hp       Previous warp file
